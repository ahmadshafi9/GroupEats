A React Native mobile application that helps friends discover and share restaurant experiences together. Built with Expo, Firebase, and modern mobile development practices.

## Features

- **Social Feed**: Share restaurant experiences with location tags and photos
- **Place Discovery**: Find restaurants with Google Maps integration and autocomplete
- **User Profiles**: Create profiles and connect with friends
- **Real-time Updates**: Firebase-powered live feed updates
- **Image Uploads**: Capture and share restaurant photos
- **Location Services**: Integrated location-based restaurant discovery

## Tech Stack

- **Frontend**: React Native with Expo & Expo Router
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **Maps**: Google Maps & Google Places API
- **Language**: TypeScript
- **Styling**: React Native StyleSheet
